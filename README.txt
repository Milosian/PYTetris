# Gra Tetris 2D w języku Python

# JAK URUCHOMIĆ?
- otwórz plik main.py lub main.exe

INSTRUKCJA GRY:
- strzałka w lewo: przemieszcza bloczek w lewo
- strzałka w prawo: przemieszcza bloczek w prawo
- strzałka w dół: przyspiesza spadanie bloczku
- klawisz C: zamiana obecnego bloczka na kolejny
- gdy przegrasz rozgrywkę naciśnij dowolny klawisz, by ponownie rozpocząć rozgrywkę
